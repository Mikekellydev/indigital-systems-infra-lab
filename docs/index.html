🚀 Quickstart

Run this lab on Ubuntu, WSL2, or any Linux environment.

Install Ansible

sudo apt update
sudo apt install ansible -y

Install Terraform

sudo apt install wget gpg -y
wget -O- https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update
sudo apt install terraform -y

Run Ansible Playbooks

ansible-playbook ansible/playbooks/baseline.yml -i ansible/inventories/dev/hosts.ini
ansible-playbook ansible/playbooks/webserver.yml -i ansible/inventories/dev/hosts.ini
ansible-playbook ansible/playbooks/database.yml -i ansible/inventories/dev/hosts.ini

🧱 Proxmox IaC (Terraform)

Infrastructure-as-code for VM provisioning using cloud-init templates.

cd proxmox/terraform
terraform init
terraform validate
terraform plan

Note: The public repo uses example values for Proxmox credentials, IPs, and templates. Replace them in your private copy before running terraform apply.

📊 Monitoring (Prometheus + Grafana)

Prometheus scrapes metrics from Linux VMs; Grafana visualizes system health.

Start Prometheus

docker run -d -p 9090:9090 \
  -v $(pwd)/monitoring/prometheus/prometheus.yml:/etc/prometheus/prometheus.yml \
  prom/prometheus

Start Grafana

docker run -d -p 3000:3000 grafana/grafana

Then open:

  • Prometheus → http://localhost:9090
  • Grafana → http://localhost:3000

🚀 Quickstart

Run this lab on Ubuntu, WSL2, or any Linux environment.

Install Ansible

sudo apt update
sudo apt install ansible -y

Install Terraform

sudo apt install wget gpg -y
wget -O- https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update
sudo apt install terraform -y

🧩 Architecture Overview

flowchart TD TF[Terraform IaC] --> PVE1(Proxmox Node 1) TF --> PVE2(Proxmox Node 2) PVE1 --> VM1[Linux VM 1] PVE2 --> VM2[Linux VM 2] Ansible[Ansible Automation] --> VM1 Ansible --> VM2 VM1 --> PROM[Prometheus] VM2 --> PROM PROM --> GRAF[Grafana]

🧱 Proxmox IaC (Terraform)

cd proxmox/terraform
terraform init
terraform validate
terraform plan

📊 Monitoring (Prometheus + Grafana)

Start Prometheus

docker run -d -p 9090:9090 \
  -v $(pwd)/monitoring/prometheus/prometheus.yml:/etc/prometheus/prometheus.yml \
  prom/prometheus

Start Grafana

docker run -d -p 3000:3000 grafana/grafana

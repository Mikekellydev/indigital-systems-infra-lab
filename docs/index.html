🚀 Quickstart (EL9 Environment)

Validated on AlmaLinux 9.4 (Seafoam Green) - RHEL 9 Binary Compatible.

1. Core Toolchain Installation

# Enable EPEL and install Ansible
sudo dnf install epel-release -y
sudo dnf install ansible git -y

# Install Terraform (HashiCorp RHEL Repo)
sudo dnf install -y yum-utils
sudo yum-config-manager --add-repo https://rpm.releases.hashicorp.com/RHEL/hashicorp.repo
sudo dnf install terraform -y

2. Podman Setup (Rootless Containers)

# Install native container tools
sudo dnf install -y container-tools
podman --version

Log Ingestion Pipeline

  -v /var/log/audit_findings.log:/var/log/audit_findings.log:Z \
  splunk/universalforwarder:latest

# Deploying Prometheus with SELinux Context
